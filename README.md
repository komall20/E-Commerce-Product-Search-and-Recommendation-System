# E-Commerce-Product-Search-and-Recommendation-System
A Flask-based web application that enables users to search for products and receive personalized recommendations using NLP and machine learning techniques.
Key Features

Search Engine: Retrieves relevant products using NLP techniques (tokenization, stemming, stopword removal) and similarity-based ranking (TF-IDF & cosine similarity).
Recommendation Module: Suggests related items based on product features, titles, and descriptions using content-based filtering.
Real-Time Interaction: Users can search and receive recommendations interactively through a user-friendly web interface.
Data Preprocessing: Engineered features from raw product data to enhance the accuracy and relevance of search and recommendations.
Exploratory Data Analysis: Conducted EDA in Jupyter notebooks to identify trends and optimize search/recommendation strategies.
Tech Stack

Backend: Flask
- NLP & ML: NLTK, scikit-learn (TF-IDF, Cosine Similarity)
- Data Analysis: Pandas, Jupyter Notebooks
- Frontend: HTML, CSS, JavaScript

How to Run
1. Clone the repository
2. Install dependencies: pip install -r requirements.txt
3. Run the Flask app: python app.py
4. Open your browser to http://localhost:5000
